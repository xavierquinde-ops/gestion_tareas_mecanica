Gestión de Tareas Mecánica

Ejercicio de sistema de gestión académica desarrollado en Python para estudiantes de Ingeniería Mecánica.

Descripción

Este proyecto permite gestionar estudiantes, tareas y operaciones básicas de un sistema académico de manera sencilla.

El sistema fue desarrollado como práctica de programación en Python utilizando conceptos básicos e intermedios.

Funcionalidades

Registro de estudiantes

Gestión de tareas

Operaciones matemáticas

Manejo de archivos CSV y JSON

Uso de funciones y condicionales

Manejo de excepciones

Uso de listas y bucles

Tecnologías utilizadas

Python 3

Git

GitHub

Cómo ejecutar el proyecto

Descargar o clonar el repositorio.

Abrir la carpeta en Visual Studio Code.

Ejecutar el archivo principal.


Ejemplo:

python main.py
